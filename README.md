# Aurora 2026 Website

A stunning, modern website for the Aurora 2026 inter-university tech conference on Social Engineering and Agentic AI featuring a dark theme with glowing orange accents.

## Design Features

- **Dark Mode Base**: Black/Deep Void (neutral-950) as the canvas
- **Glowing Accents**: Amber-500 to Orange-400 gradients creating an "Aurora" effect
- **Glassmorphism**: Frosted glass effects on navigation and cards
- **Modern Layout**: Clean grid layouts inspired by tech competition sites

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **Lucide React** for icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
